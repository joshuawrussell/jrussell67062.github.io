Several months ago I ran across an article published by Scott Vinkle (Medium). It was interesting to see how Github could be used to host static (more on this in a moment) websites for free. It is a great tutorial - I thought I would extend it and provide a few more steps to get a bootstrap template setup and an intial website working.

I mentioned using github for static websites because this would not be viable solution for large data driven or ecommerce sites, but for sites like this one where all of the content is manually generated and only a place to host the content is needed - it works well. HTML, CSS and Javascript are supported. This solution would be adequate for a blog or a portfolio website.

For this tutorial, we will use Microsoft Code as an editor. Microsoft Code has built in integration for GIT and extensions that will allow us to create, modify and view our local webiste in a light weight web server.

3. Setting up the Bootstrap Template

You might be thinking - what is a Bootstrap or what is a Bootstrap template? Bootstrap is a layout tool that provides a consistent, modern, responsive framework that will allow us to get a working website without spending alot of time working out inonsistencies between different browsers (Internet Explorer, Edge, Firefox, Chrome).

You can find many basic templates here: Bootstrap Templates

There are many Bootstrap Templates to choose from and many are free. For this tutorial, we will be using parts of the Modern Business template. You can see a preview of it here: Modern Business Preview. You can download and explore the files if you would like, but we will not need the files for this tutorial.
